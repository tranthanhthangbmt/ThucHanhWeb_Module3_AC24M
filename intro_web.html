Quy Trình 7 Bước Xây Dựng Web Fintech

1Cài Đặt Công Cụ AI

Sử dụng Google Antigravity (hoặc Cursor/Windsurf) để hỗ trợ viết code tự động.

Tải Google Antigravity

2Chọn Đề Tài (Persona)

Sinh viên chọn 1 trong 4 nhóm đề tài bên dưới để thực hiện đồ án môn học.

1. TaxAlert - Trợ Lý Pháp Lý Tin tức & Cảnh báo

Persona: Kế toán thuế sợ bỏ sót văn bản luật mới.

Yêu cầu: Dashboard tin tức, lịch nộp thuế đếm ngược, phân loại văn bản (GTGT, TNDN).

2. SmartBooks - Kế Toán Cho Startup Dịch vụ & Landing Page

Persona: Startup cần dịch vụ kế toán tiết kiệm, minh bạch.

Yêu cầu: Bảng giá so sánh (Thuê ngoài vs Nội bộ), Form đăng ký tư vấn, Case study khách hàng.

3. FinRate Hub - Sàn Tỷ Giá Data Dashboard

Persona: Kế toán thanh toán cần theo dõi tỷ giá USD/EUR để tối ưu dòng tiền.

Yêu cầu: Biểu đồ biến động, Ticker chạy giá, Máy tính quy đổi ngoại tệ.

4. E-Invoice Checker - Soi Hóa Đơn Tiện ích & Audit

Persona: Kế toán công nợ lo sợ nhận phải hóa đơn của doanh nghiệp bỏ trốn.

Yêu cầu: Tra cứu MST, Upload XML kiểm tra, Danh sách Blacklist rủi ro.

3Viết Prompt (Ra Lệnh Cho AI)

Copy các Prompt mẫu dưới đây vào khung chat của AI để tạo giao diện ban đầu.

Mẫu Prompt Tạo Giao Diện (Ví dụ cho Topic 3 - FinRate Hub):
"Tôi muốn tạo một Dashboard theo dõi tỷ giá ngân hàng có tên là 'FinRate Hub'. Yêu cầu: - Giao diện: Dark Mode chuyên nghiệp (màu đen, xám, xanh neon). - Bố cục: Có Sidebar bên trái, phần nội dung chính bên phải. - Nội dung chính: + Một thanh trượt (Ticker) hiển thị tỷ giá USD, EUR, JPY. + Hai biểu đồ (Chart.js): Một biểu đồ đường thể hiện biến động USD trong 30 ngày, một biểu đồ cột thể hiện lãi suất. + Một bảng so sánh tỷ giá giữa Vietcombank, Techcombank, ACB. - Viết code HTML và CSS (trong file style.css riêng) đầy đủ và đẹp mắt."

4Tạo Nội Dung Chuyên Ngành (Content AI)

Sử dụng AI để viết nội dung "thật" và chuyên nghiệp, thay vì dùng văn bản giả (Lorem Ipsum).

Prompt Tạo Nội Dung (Topic 1 - TaxAlert):
"Đóng vai chuyên gia tư vấn thuế, hãy viết một đoạn cảnh báo ngắn (khoảng 50 từ) về mức phạt chậm nộp tờ khai thuế GTGT theo Nghị định 125. Giọng văn nghiêm khắc nhưng mang tính nhắc nhở hỗ trợ."
Prompt Tạo Chính Sách Bảo Mật (Topic 4 - Invoice Check):
"Viết một cam kết bảo mật dữ liệu cho người dùng khi họ upload file XML hóa đơn lên hệ thống. Cam kết không lưu trữ dữ liệu, tuân thủ Luật An ninh mạng, giọng văn tạo sự an tâm tuyệt đối."

5Mẹo Nâng Cao: Cào Dữ Liệu Tự Động (Data Scraping)

Dành cho các nhóm muốn lấy dữ liệu thật từ website ngân hàng (Ví dụ: Vietcombank, General Dept of Taxation).

Prompt Cho Google Antigravity:
"Hãy viết cho tôi một đoạn mã Python đơn giản sử dụng thư viện BeautifulSoup để lấy dữ liệu tỷ giá USD từ trang web của Ngân hàng Vietcombank. Sau đó, hãy hướng dẫn tôi cách tích hợp hoặc hiển thị dữ liệu này lên trang web HTML mà tôi đang xây dựng."

*Lưu ý: AI sẽ hướng dẫn bạn cách chạy script Python để lấy data và lưu vào file JSON/CSV để web đọc.

6Đưa Lên GitHub (Nộp Bài)

  1. Tạo tài khoản tại GitHub.com.
  2. Tạo New Repository.
  3. Upload toàn bộ thư mục dự án (index.html, style.css, script.js, images...) lên.
  4. Vào Settings -> Pages, chọn Branch main và Save.
  5. Copy đường link (VD: username.github.io/my-project) để nộp cho giảng viên.
